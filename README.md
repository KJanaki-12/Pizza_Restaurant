# 🍕 Responsive Restaurant Website

A modern, responsive, and accessible restaurant website built using **HTML, CSS, Bootstrap, Flexbox, and CSS Grid**.  
This project includes a responsive navbar, hero section, about section, menu, Bootstrap carousel, gallery, Google Maps integration, and footer.

# 📌 Project Overview

This project is designed to create a professional restaurant website UI that works smoothly across all devices while following accessibility best practices.

The webpage includes:

- Responsive Header/Navbar
- Hero Banner Section
- About Us Section
- Restaurant Menu Section
- Bootstrap Carousel
- Responsive Image Gallery
- Find Us Section with Google Maps
- Footer Section
- Mobile Responsive Design
- Accessibility Features
